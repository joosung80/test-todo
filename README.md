# Todo App - Glassmorphism Design

아이폰 글래스모피즘 스타일을 적용한 Next.js Todo 웹 애플리케이션입니다.

## 기능

- ✅ Todo 항목 추가
- ✅ Todo 항목 완료/미완료 토글
- ✅ Todo 항목 삭제
- 💾 로컬 스토리지에 자동 저장

## 기술 스택

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- 로컬 스토리지

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

### 빌드

```bash
npm run build
```

### 프로덕션 실행

```bash
npm start
```

## 디자인

아이폰 글래스모피즘 디자인을 적용했습니다:

- 반투명 배경 (`backdrop-blur`)
- 부드러운 그림자 효과
- 애니메이션 그라데이션 배경
- 둥근 모서리
- 미묘한 테두리

## 배포

이 프로젝트는 Vercel, Netlify 등 다양한 플랫폼에 배포할 수 있습니다.
